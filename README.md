# Star Visibility

In the <tt>staralt.py</tt> file, check the inputs and input a location, target(s), and observing date.

In the terminal, run:

<center><tt>python staralt.py</tt></center>

to see star visibility for the chosen observing night.

To see visibility for the current date, run:

<center><tt>python staralt.py < now / today / tonight ></tt></center>
